Słowem wstępu

JavaScript jest potężnym językiem w którym możesz tworzyć nie tylko aplikacje, które będą działać po stronie klienta ale również po stronie serwera. Przedstawione tutaj przykłady nie zawierają pewnie nawet 1/10000 tego co oferuje nam JS.

Jest to język lekki w nauce, ale ucząc się go potrzeba dużo cierpliwości i samozapracia.

Ten maluczki projekt nie miał na celu pokazywania miliona przkładów jak można wykorzstać JavaScript, wręcz przeciwnie. Przedstawione króciutkie przykłady mają pokazać jak wielki potencjał drzemie w tym języku, który sam powinieneś odkryć.

Zdarzenia

Wchodząc na strone internetową, użytkownik wykonuje wiele czynności takich jak: kliknięcie, przesunięcie myszą, zaznaczenie elementu, wcisnięcie klawisza i wiele innych. JavaScript pozwala Ci w pełni kontrolować te zdarzenia.

Istnieją rwónież zdarzenia, które nie są spowodowane przez interakcje np. DOMContentLoaded.

const block = document.querySelector(".block"); block.addEventListener("mousemove", function(e){   const el = document.querySelector(".element"),         x = e.pageX - this.offsetLeft,         y = e.pageY - this.offsetTop;   el.style.left = x + "px";   el.style.top = y + "px"; })

Tworzenie i usuwanie elementów

JavaScript umożliwia nam tworzenie nowych elementów na stronie jak i ich usuwanie. Nowo utworzonymi elementami można dowolnie manipulować, przypisywać im id, klasy, style, atrybuty, możemy również decydować jaką będą posiadały zawartość.

Nowe elementy mogą być umieszczane przed, za elementami i wewnątrz innych elementów.

function createElement(){   const newEl = document.createElement("div"),         parent = document.querySelector(".el");   newEl.innerHTML = "Dodano nowy element";   par.appendChild(newEl); }
function removeChild(){   const parent = document.querySelector(".el1"),         child = document.querySelector(".el2");   if(child !== null){      parent.removeChild(child);   } }

Manipulacja elementami

Cóż, JavaScript jest tak świetnym językiem, że pozwala nam nawet z prostokąta zrobić kwarat a z kwadratu koło. Elementom można przypisywać/dopisywać nowe atrybuty, klasy, style, które mogą zmienić jego wygląd, zawartość czy też sprawić, że element stanie się kompletnie niewidoczny.

function change(){   const el = document.querySelector(".block");   if(!el.classList.contains("rectangle") &&      !el.classList.contains("circle")){     el.classList.add("rectangle");     el.innerHTML = "Ale jak to..."   } else {     el.classList.remove("rectangle");     el.classList.add("circle"); }
Jestem białym kwadratem!

Animacje

Dużą część animacji można wykonać już w CSS, ale JavaScript pod tym wzlędem również ma duże pole do popisu.

function fadeOut(){   const el = document.querySelector(".block");   el.style.opacity = +(el.style.opacity ||     window.getComputedStyle(el)["opacity"]);   (function task(){     el.style.opacity = +el.style.opacity-0.01;     if(+el.style.opacity>0){       (window.requestAnimationFrame &&        requestAnimationFrame(task)) ||        setTimeout(task, 16);     }   }()); }
